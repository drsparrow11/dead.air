# DEAD.AIR

Static GitHub Pages site for **DEAD.AIR | The Vox Chronicles** by SPARROW x ECHO.

## Contents

- `index.html` - full album site
- `styles.css` - responsive visual system
- `script.js` - data-driven track player, lyrics, and gallery
- `assets/audio/*.mp3` - MP3 exports converted from the provided WAV files
- `assets/images/*.png` - album and track imagery
- `robots.txt`, `sitemap.xml`, `llms.txt`, `humans.txt`, `site.webmanifest`

## Deploy

Upload these files to the root of `drsparrow11/dead.air`, then enable GitHub Pages from the `main` branch root.
